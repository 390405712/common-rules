## 安装

- **步骤 1：** 安装依赖

  ```bash
    # 选择一个你喜欢的包管理器

    # NPM
    $ npm install common-rules --save

    # Yarn
    $ yarn add common-rules

    # pnpm
    $ pnpm install common-rules
  ```

- **步骤 2：** 引入正则

    ```ts
    // 例如
    import { RegExpMobilePhoneNunber } from 'common-rules'
    ```
